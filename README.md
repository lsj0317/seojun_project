# typing_game

## 랜덤한 영어단어를 입력하면서 시간내에 점수를 쌓는 게임입니다

### 게임시작 누르면 10초의 카운트가 시작됩니다
### 단어를 맞게 입력하면 다음단어로 넘어갑니다 / 시간이 다시 10초로 카운트됩니다 / 점수가 증가합니다
### 게임중에는 게임시작버튼이 게임중으로 표시됩니다
### 10초의 시간이 지나면 게임이 종료되고 처음시작으로 돌아옵니다
